# Arduino Temperature Monitoring System

This project monitors temperature using a TMP36 sensor and Arduino Uno.

## Components
- Arduino Uno
- TMP36 Temperature Sensor
- LED
- 220Ω Resistor

## Features
- Reads temperature from sensor
- Displays temperature in Serial Monitor
- LED turns ON when temperature exceeds threshold

## Project Description
Designed a temperature monitoring system using Arduino Uno and TMP36 sensor.

Implemented analog sensor data acquisition and temperature calculation.

Developed threshold-based alert using LED indicator.
